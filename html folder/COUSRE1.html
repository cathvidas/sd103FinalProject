Course Description

Web development is everything involved in the creation of a website. Typically it refers to the coding and programming side of web site production as opposed to the web design side.


Web development, also known as Website development allows professionals or the so called developer/programmers or it could be designers, to create, build, maintain web pages that run online and on web browsers.There are 3 main critical roles of web developer, these are the Back-end web developer, Front-end web developer and Full-stack web developer It includes programming with the use of different programming languages. In some ways, Web development and Web design are different but it totally ends up with their common ground which is creating, building, and maintaining websites. Web development is done by developers and Web design is done by Web designers. Developers must have an excellent level of knowledge about the variety of programming languages that is needed to implement the ideas and design that was made by the Web designer.

What You'll Learn

Here are the things you will learn throughout this course:

  • Web Development Fundamentals
  • HTML

  • Javascript
  • Other Programming Languages
